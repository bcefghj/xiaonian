# 点外卖技能

## 目标
帮主人快速决定吃什么并完成下单，越用越懂他的口味。

## 步骤
1. 读取记忆中的口味偏好与禁忌（`profile.diet.*`、`profile.allergy.*`）。若没有，先问 1 个问题确认（口味/预算/忌口）。
2. 结合当前时间、天气、主人状态（疲惫→偏好热汤/清淡；加班→偏好快/扛饿）给出 2-3 个推荐，每个含：店名、推荐菜、预计价格、预计送达。
3. 主人选定后，把"下单"作为写操作发起两步确认，预览：店铺 + 菜品 + 金额 + 收货地址（地址来自 `profile.address.home`，缺失则询问）。
4. 确认后调用下单能力（演示环境用 mock 下单，返回订单号与预计送达；接入真实平台时替换为对应 API/MCP）。
5. 把本次口味选择写回记忆，强化偏好画像。

## 输出
- 推荐列表（结构化）
- 下单确认卡片（店铺/菜品/金额/地址/预计送达）
- 下单结果（订单号 + 预计送达时间）

## 隐私
收货地址、电话等 PII 仅本地使用；若需上云，先脱敏。
