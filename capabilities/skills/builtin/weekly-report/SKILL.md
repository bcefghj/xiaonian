# 写周报技能

## 目标
把零散的工作记录整理成一份清晰的周报，主人只需验收。

## 步骤
1. 收集素材：最近一周的对话要点、完成的任务（审计日志/记忆）、相关文件（可用电脑能力读取指定目录）。
2. 归类：按"本周完成 / 进行中 / 下周计划 / 风险与需要的支持"四块组织。
3. 生成正文：条目化、量化（带数字/结果），语气专业克制。
4. 询问是否写入本地文件（如 `~/Desktop/周报_YYYYMMDD.md`）；写文件为写操作，走两步确认。
5. 可选：导出为 PDF（调用本地排版能力）。

## 模板
见同目录 `template.md`（资源级，按需读取）。

## 输出
- 周报 Markdown 正文
- （确认后）写入的本地文件路径
