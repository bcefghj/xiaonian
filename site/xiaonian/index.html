技术方案
企业框架优先，每个技术点都有背书

不从零造轮子：底层全部站在大厂在生产验证过的成熟开源框架上，我们只写产品逻辑。

编排 LangGraph持久执行 + 人在环 + 状态持久化；Klarna / Uber / 摩根大通在生产使用。
记忆 Mem0业界领先的 Agent 记忆层，亚秒级、可本地；记忆向量化在本机完成，内容不出域。
技能 Agent SkillsAnthropic 开放标准，三级渐进式披露；Atlassian / Figma / Canva / Stripe / Notion 采用。
电脑操控受控 harness：读操作即时、写操作两步确认；可切换 Open Interpreter 原生沙箱。
模型层多 provider 可切换：MiniMax M2 / DeepSeek / 小米 MiMo（OpenAI + Anthropic 双协议）。
安全隐私两步确认 + PII 上云前脱敏 + 本地审计日志 + 数据/凭证全本地。
