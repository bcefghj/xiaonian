# ADR-0004：能力扩展用 Anthropic Agent Skills 开放标准

- 状态：已采纳
- 日期：2026-06-30

## 背景
小念核心只保留四件事（懂你 / 主动关心 / 干活 / 进化），但用户的"杂"需求无穷
（点外卖、打车、写周报、查天气、家居控制…）。需要一种"核心不臃肿、能力可无限生长"的机制。

## 决策
采用 Anthropic Agent Skills 开放标准：每个技能是一个目录，含 `SKILL.md`
（YAML 元数据 + Markdown 指令），可带脚本/模板等引用文件。

三级渐进式披露（progressive disclosure）：
1. 启动级：只把"名称 + 描述"载入上下文（每技能几十 token），核心永不臃肿。
2. 触发级：命中某技能时才载入它的 SKILL.md 正文。
3. 资源级：正文引用的脚本/模板，用到才读。

新增能力 = 新增一个目录，核心逻辑零改动；技能可跨平台移植（同标准被
Atlassian/Figma/Canva/Stripe/Notion 等采用）。

## 结果
- 正：故事线清晰（核心精简 + 按需进化）；可做 Skill 市场，生态可复用。
- 负：技能质量依赖 SKILL.md 编写规范（已提供 3 个示范技能作为模板）。
