# ADR-0002：本地优先、隐私主权

- 状态：已采纳
- 日期：2026-06-30

## 背景
小念定位是"住在你电脑里的私人秘书"，要管理用户电脑里的事，必然触及大量隐私。
隐私是相对云端助手（如豆包）的核心差异化卖点。

## 决策
1. 数据全本地：记忆（Mem0 本地 Qdrant）、结构化画像（SQLite）、审计/trace 日志、
   凭证，全部存于本机 `data/`，默认不出域。
2. 本地隐私嵌入器：记忆向量化在本机完成（`memory/embedder.py`），
   记忆内容不发往任何云端 embeddings 服务。
3. 上云最小化 + 脱敏：仅把"最小必要上下文"发给云端 LLM，且发送前对手机号/邮箱/
   身份证/银行卡做 PII 脱敏（`security/redact.py`）。
4. 算力本地：电脑操控直接调用本机算力执行任务。

## 结果
- 正：隐私可讲清、可演示；契合"管家"定位；企业内网/私有化友好。
- 负：本地嵌入器召回质量低于大模型 embeddings（已在架构上保留一键替换为
  fastembed / OpenAI 兼容嵌入端点的能力）。
